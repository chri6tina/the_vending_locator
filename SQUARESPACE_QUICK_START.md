# 🚀 Squarespace Migration - Quick Start Guide

## ⚡ Immediate Next Steps

### 1. Sign Up for Squarespace (5 minutes)
1. Go to [squarespace.com](https://www.squarespace.com)
2. Click "Get Started"
3. Choose **Business Plan** ($26/month) - recommended for custom CSS
4. Select any template (we'll customize it)

### 2. Set Up Your Domain (5 minutes)
- **Option A**: Use free Squarespace domain (yourbusiness.squarespace.com)
- **Option B**: Connect your existing domain if you have one

### 3. Create Your Pages (15 minutes)
1. **Home** - Main landing page
2. **Services** - Your vending machine location services
3. **Pricing** - Service packages and pricing
4. **Blog** - Content marketing
5. **Contact** - Contact form and information

### 4. Apply Green Theme (10 minutes)
1. Go to **Settings** → **Advanced** → **Code Injection**
2. Copy the entire content from `squarespace-green-theme.css`
3. Paste it into the **CSS** section
4. Click **Save**

### 5. Import Your Content (30 minutes)
1. **Homepage**: Copy content from your `index.html`
2. **Services**: Copy from `services.html`
3. **Pricing**: Copy from `pricing.html`
4. **Contact**: Copy from `contact.html`

## 🔧 Technical Setup

### Custom CSS Injection
1. **Settings** → **Advanced** → **Code Injection**
2. **CSS** tab
3. Paste the entire `squarespace-green-theme.css` content
4. **Save**

### SEO Setup
1. **Settings** → **SEO**
2. Add page titles and descriptions
3. Set up Google Analytics
4. Add alt text to images

### Form Setup
1. **Contact page** → **Add Form Block**
2. Configure form fields
3. Set up email notifications
4. Test form submission
